# Claude Web Bootstrap

> Claude Web (чат claude.ai) не читает файлы репозитория автоматически.  
> Универсальные правила — в `SYSTEM.md`. Загрузи вручную.

## Как загрузить систему

**Project Instructions (рекомендуется):**

1. claude.ai → Projects → создать проект (Project)
2. Вставить содержимое `SYSTEM.md` в поле «Проект инструкции»

**Или в начале каждого чата:**

Сказать Claude: «Вот мои правила работы: [SYSTEM.md]»

## Ограничения

- Нет автоматического чтения файлов репозитория
- Нет git-интеграции
- Нет персистентной памяти между сессиями (без Projects)
- Нет запуска скиллов через `skill:` — скиллы включаются через описание в запросе

## Context handling

- Вставлять только нужный фрагмент, не весь файл
- Указывать путь к файлу перед блоком кода
- При длинных сессиях — напоминать ключевые decisions из `decisions.md`

## Что работает

- Анализ кода (вставить фрагмент)
- Генерация кода по spec
- Review дифов
- Архитектурное обсуждение
- Skills из `SKILLS_ALL.md` (вставить в Project Instructions)
