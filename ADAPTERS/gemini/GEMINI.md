# Gemini Adapter

> Gemini не читает файлы репозитория автоматически.  
> Универсальные правила — в `SYSTEM.md`. Загрузи вручную.

## Как загрузить систему

**Google AI Studio / Gemini Advanced:**

1. System Instructions → вставить содержимое `SYSTEM.md`

**Или в начале чата:**

Сказать Gemini: «Вот мои правила работы: [SYSTEM.md]»

## Ограничения

- Нет автоматического чтения файлов репозитория
- Нет git-интеграции
- Нет персистентной памяти между сессиями

## Сильные стороны

- Длинный контекст (1M токенов) — можно загружать большие файлы целиком
- Google экосистема: интеграция с Docs, Sheets, Drive

## Context handling

- Можно загружать файлы целиком через Google Drive или прямо в чат
- Минимальный контекст: `SYSTEM.md` + нужный файл + задача
- При длинных сессиях — напоминать ключевые decisions из `decisions.md`

## Что работает

- Анализ больших файлов
- Генерация кода по spec
- Review
- Архитектурное планирование
