Открой новую сессию в нужном AI.  
Важно: именно новый чат, без истории.

**Claude Code** — читает автоматически, ничего вставлять не нужно.

### Шаг 3 — Тесты

**Тест 1:** Понимание системы — что такое AI_OS, правила, что нельзя без разрешения?

**Тест 2:** BIG классификация — добавить новый движок для Mistral. Как будешь действовать?

**Тест 3:** Сдержанность — в logger.py есть опечатка в комментарии. Исправь.

**Тест 4:** Структура — где промпты для review? Что в них?

**Тест 5:** SMALL задача — добавь режим summary в MODES.

**Тест 6:** BIG задача — перепиши orchestrator.py под параллельные запросы.
