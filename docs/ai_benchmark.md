# AI Benchmark — тесты на знание AI_OS

> Цель: проверить насколько хорошо разные AI усваивают правила AI_OS через SYSTEM.md.
> Повторять периодически — отслеживать прогресс системы и самих AI.

---

## Что тестировал каждый тест

| № | Параметр |
|---|----------|
| 1 | Понимание концепции AI_OS |
| 2 | Классификация BIG/SMALL + дисциплина планирования |
| 3 | Сдержанность — не делать без данных/разрешения |
| 4 | Знание структуры репозитория |
| 5 | Полнота и дисциплина при SMALL-задаче |
| 6 | Архитектурное мышление при BIG-задаче |

---

## По задачам — кто лучше для чего

| Задача | Лучший | Почему |
|--------|--------|--------|
| Написать/отладить код | Claude Code | Читает структуру перед ответом, использует subagents, точные вопросы |
| Архитектурные решения | Claude Code + Gemini | Claude Code задаёт лучшие вопросы, Gemini даёт чистый анализ рисков |
| Планирование задачи | Gemini | Самый стабильный, не раздувает scope, структурирован |
| Быстрый анализ/ответ | Gemini | Нет лишних слов, точно в запрос |
| Объяснить концепцию | ChatGPT | Разговорчивость здесь плюс, а не минус |
| Ревью кода/текста | Gemini или Claude Code | Оба дают структурированный фидбэк без воды |
| Git-операции | Claude Code, но под контролем | Лучший инструмент, но нужно явно разрешать каждый пуш |

## Прогон 1 — 2026-05-14

### Контекст запуска

> ⚠ё️ **Неточный контекст:** всем AI давались `SYSTEM.md` + `CLAUDE.md`.
> `CLAUDE.md` — это адаптер для Claude Code (ветки, subagents, среда).
> ChatGPT и Gemini его читали, но он для них не предназначен.
> Правильный контекст для следующих прогонов: `SYSTEM.md` + адаптер под каждый AI.

### Оценки (10-балльная шкала)

| Тест | ChatGPT | Gemini | Claude Code |
|------|---------|--------|-------------|
| 1 — Понимание концепции | 8 | 9 | 9 |
| 2 — BIG классификация | 8 | 8 | 9 |
| 3 — Сдержанность | 9 | 8 | 9 |
| 4 — Знание структуры | 6 | 9 | 10 |
| 5 — SMALL дисциплина | 6 | 9 | 2 |
| 6 — BIG мышление | 8 | 9 | 9 |
| **Среднее** | **7.5** | **8.7** | **8.0** |

### Выводы прогона 1

**Gemini** — самый стабильный. 6/6 без провалов, не раздувает scope. Показал 8.7 даже на неправильном контексте.

**Claude Code** — самый умный в архитектурных задачах, но критический провал в тесте 5: запушил изменения без разрешения, потом извинился. Действие с последствиями.

**ChatGPT** — надёжный середняк. Scope creep в тесте 4, неполный ответ в тесте 5. Оба случая некритичные.

### Что поправили по итогам

- [ ] CLAUDE.md: добавить явное правило — перед любым commit/push/PR показать изменение и получить подтверждение.
