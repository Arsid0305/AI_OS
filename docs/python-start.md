# Памятка запуска Python

## Шаг 1 — открыть cmd

`Win + R` → ввести `cmd` → Enter

## Шаг 2 — перейти в папку runtime

```
cd C:\DATA\AI_OS\runtime
```

## Шаг 3 — активировать venv

```
venv\Scripts\activate
```

Если всё правильно, строка станет:

```
(venv) C:\DATA\AI_OS\runtime>
```

> Каждый раз при открытии cmd — venv нужно активировать заново. Это нормально.
