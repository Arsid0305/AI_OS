# Domain Rules: VISUAL

- Варианты A/B должны различаться по АКЦЕНТУ, не только по формулировке
- Указывать требования платформы: соотношение сторон, text-safe зоны
- Compliance check обязателен: запрещённый контент, brand safety
- Не генерировать изображения — только промпты для генераторов
- Forbidden: варианты, отличающиеся только синонимами
