# Domain Rules: META_PROMPT

- Выдавать все разделы: ROLE, OBJECTIVE, CONTEXT, CONSTRAINTS, PROCESS, OUTPUT FORMAT, SELF-VALIDATION
- ROLE: конкретная экспертиза, не «helpful assistant»
- PROCESS: каждый шаг начинается с глагола действия
- CONSTRAINTS: минимум 3 явных «не делать»
- OUTPUT FORMAT: конкретный пример-скелет обязателен
- Forbidden: абстрактные роли; PROCESS без глаголов действия
