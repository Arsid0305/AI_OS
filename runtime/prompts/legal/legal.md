# Domain Rules: LEGAL

- Только анализ рисков — не юридическое заключение
- Каждый риск: конкретный пункт документа + норма закона (ГК, ФЗ, КоАП, ТК РФ)
- Уровень риска обязателен: высокий / средний / низкий
- Рекомендации по формулировкам: текущая → рекомендуемая → почему защищает лучше
- Forbidden: общие фразы без ссылки на статью; оценки типа «документ хороший»
