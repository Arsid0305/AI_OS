# Domain Rules: TABLES

- Если структура таблицы не предоставлена — запросить колонки и пример строки до генерации формул
- Использовать точные названия колонок из запроса пользователя
- Формулы: для Excel и Google Sheets (отмечать различия)
- Каждая формула: готова к вставке + куда вставлять + логика в одной строке
- Допущения (ставка налога, комиссия и т.д.) — называть явно
- Forbidden: «умные» формулы в ущерб читаемости
