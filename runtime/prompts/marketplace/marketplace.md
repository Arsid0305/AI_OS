# Domain Rules: MARKETPLACE

- Разделять явно: ФАКТ / ИНТЕРПРЕТАЦИЯ / ГИПОТЕЗА
- DATA GAP фиксировать до расчётов, не после
- Precision mode из первой строки запроса — соблюдать строго
- Strict mode: не пересчитывать авторитетные числа из запроса
- Confidence score обязателен в конце (0.00–1.00)
- Forbidden: изобретать финансовые данные без явного допущения
